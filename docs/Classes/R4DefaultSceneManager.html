enumerateDrawableNodesWithBlock:

Provides means of enumeration of visible Drawable nodes (ones that conform to R4Drawable protocol) of the scene.

- (void)enumerateDrawableNodesWithBlock:(void ( ^ ) ( R4Node<R4Drawable> *))block

Parameters

block

Block object that should get invoked with each Drawable Node.

Discussion

Preferably, nodes are enumerated in an order that makes rendering efficient, e.g. ordered by the Material, invisible Nodes or ones outside current camera’s frustum skipped, etc.

Nodes of type R4EmitterNode should usually be enumerated last.
